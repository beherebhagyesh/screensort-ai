# Tech Stack

## Core Technologies
-   **Backend (Web):** Node.js with Express framework.
-   **Backend (Automation):** Python 3.
-   **OCR:** Tesseract OCR (interfaced via `pytesseract`).
-   **Image Handling:** PIL (Pillow).

## Infrastructure & Platform
-   **Primary Environment:** Android (Termux).
-   **Roadmap:** iOS support (cross-platform compatibility for paths and dependencies).
-   **Background Execution:** (Planned) cron, PM2, or a dedicated Android Service wrapper.

## Frontend
-   **Architecture:** Single Page Application (SPA) principles.
-   **Styling:** CSS3 with Material Design components (e.g., via a library or custom implementation).
-   **Web App Features:** Progressive Web App (PWA) with Service Workers and Web App Manifest.

## Dependencies
-   **Node.js:** `express`, `cors`.
-   **Python:** `pytesseract`, `Pillow`, `os`, `shutil`.
